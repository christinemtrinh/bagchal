# Versions
Node.js: v20.18.1<br />
npm: 10.8.2

# How to open Server 
Travel to Server folder and type
```
npm start
```
[Click here](http://localhost:8080/api/test) or type http://localhost:8080/api/test into a new tab to see Server


# How to open Client
Travel to Client folder and type 
``` 
npm i
npm run dev
```
